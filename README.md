A tool to quickly change the color palette of Wii Virtual Console NES games without having to use a hex editor.
## **Usage**
1. Download and unzip [windows.zip](https://github.com/dylanp14132/wii-nes-palette-tool/releases/download/v1.0/windows.zip).
2. Download and unzip [wwPacker](https://gbatemp.net/download/wwpacker.36649/) (I recommend getting v1.80 so you can skip the installer).
3. Drag your Wii VC NES WAD file onto `__wwunpacker.bat` to unpack the WAD.
4. Open `wii-nes-palette-tool.exe` and select `00000001.app` from the folder created by wwPacker as the file path.
5. Choose your color palette and click apply.
6. Repack your WAD by dragging the unpacked wad folder onto `_wwpacker-NoMod.bat` (any `_wwpacker` bat file should work but this one is recommended).
7. Voila! You now have a WAD with a new color palette.
